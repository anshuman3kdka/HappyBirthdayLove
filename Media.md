# 🎞️ Nemo Project Media Guide

This document lists all media assets required for the cinematic experience. To personalize the site, upload your files to the corresponding folders in `public/assets/` using the exact filenames listed below.

## 📂 Folder Structure
- `public/assets/image/`: All photographs and background textures.
- `public/assets/video/`: Cinematic clips and film reels.
- `public/assets/audio/`: Soundscapes, voice notes, and sound effects.
